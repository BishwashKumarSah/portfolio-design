Twitter

This Twitter clone, built with Next.js and TypeScript, features real-time updates, infinite scrolling, and an intuitive explore search. Users can authenticate via Google OAuth, post tweets, follow/unfollow others, and view profiles. The backend is powered by GraphQL, with Prisma ORM managing PostgreSQL database operations. TanStack React Query is used for efficient data fetching, while Redis improves response times by up to 40% with caching. AWS handles storage and deployment, and rate limiting ensures API security. The frontend is styled using Tailwind CSS, ensuring a responsive and modern user experience.
